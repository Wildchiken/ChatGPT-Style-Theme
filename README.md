## 功能特性

- **全面主题支持**：兼容 Cherry Studio 的亮色和暗色模式，色彩保持 ChatGPT 品牌一致性。  
- **优化消息布局**：支持长文本、代码段和连续数字显示，不会破坏布局。  
- **自适应输入框**：根据输入内容自动调整高度，空输入状态下保持紧凑单行高度。  
- **改进 Markdown 渲染**：优化标题、引用块和代码块显示，提升阅读体验。  
- **细节优化**：去除多余边框和阴影，重新设计悬浮菜单、下拉列表及滚动条样式。  
- **轻量动画效果**：消息显示平滑淡入，等待状态视觉效果简洁自然。

## 安装指南

1. 打开 Cherry Studio 客户端。  
2. 进入 **设置 → 外观 → 自定义 CSS**。  
3. 打开 `chatgpt-theme.css` 文件，将全部内容**复制粘贴**到自定义 CSS 区。  
4. 保存后主题立即生效。
