## Features

- **Full Theme Support**: Compatible with both light and dark modes in Cherry Studio, maintaining ChatGPT brand colors.  
- **Optimized Message Layout**: Supports long text, code blocks, and consecutive numbers without breaking the layout.  
- **Adaptive Input Box**: Automatically adjusts height based on input; keeps compact single-line height when empty.  
- **Enhanced Markdown Rendering**: Improved headings, blockquotes, and code block display for better readability.  
- **UI Details Refined**: Removes unnecessary borders and shadows; redesigns hover menus, dropdowns, and scrollbars.  
- **Lightweight Animations**: Smooth fade-in for messages; minimal and elegant visual effects for waiting states.

---

## Installation

1. Open Cherry Studio.  
2. Go to **Settings → Appearance → Custom CSS**.  
3. Open the `chatgpt-theme.css` file, copy all its contents, and paste it into the Custom CSS field.  
4. Save and the theme will take effect immediately.
